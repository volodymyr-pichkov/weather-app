## Description

This React app allows users to search for weather by city name. By typing a city and pressing Enter or clicking the search icon, the app fetches weather data from OpenWeatherMap and displays:

- Temperature (Â°C)  
- Humidity (%)  
- Wind speed (km/h)  
- Weather icon  
- City name
